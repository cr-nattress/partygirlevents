# US-009: How to Plan a Low-Waste Mountain Wedding (Without Sacrificing Style)

**Epic:** Epic 09 - Trend Blog Posts
**Priority:** P2
**Story Points:** 3
**Status:** Draft

## Description

Create an actionable, non-preachy blog post showing couples how to plan a sustainable mountain wedding that looks luxurious, not "crunchy." This post reframes sustainability as mainstream, aspirational, and practical—not a sacrifice.

Based on trend research (Section 6: Sustainability), this post addresses the growing expectation that weddings should be eco-conscious. Key message: Sustainability is no longer niche. It's the new default for thoughtful couples. And it can be beautiful.

Covers local florals, recycled/rented decor, low-waste catering, environmentally accredited venues, carbon-neutral travel, and digital-first planning. Features an interactive sustainability scorecard and eco-swap recommendations.

**Target Keyword:** "sustainable mountain wedding"
**Target Word Count:** 1500-1800 words
**Target Audience:** Eco-conscious couples (Gen Z, millennial), couples marrying in Colorado mountains, couples who want to reduce environmental impact without looking "cheap"

## Technical Notes

### Component Architecture
- Scorecard component:
  - React state for toggle switches and score calculation
  - Score algorithm: each item worth X points (total = 100)
  - Conditional rendering of recommendations based on unchecked items
  - Local storage to persist scorecard state
  - Share functionality (Web Share API or custom social links)
  - Email capture form for "Save My Scorecard"
- Eco-swap cards:
  - CSS 3D transforms for flip animation
  - Fallback for browsers without transform support
  - Touch-friendly on mobile
- Carbon footprint chart:
  - Animated SVG or CSS-based bar chart
  - Intersection Observer to trigger animation on scroll
  - Data-driven (values stored in config for easy updates)

### Performance Considerations
- Lazy load images and interactive components below fold
- Scorecard logic lightweight (simple addition)
- Card flip animations CSS-based (performant)
- Chart animation uses CSS transforms (GPU-accelerated)
- Target Lighthouse: 90+ performance, 100 accessibility, 100 SEO

### Data Management
- Scorecard items stored in structured format (JSON or CMS):
  - Item name
  - Category
  - Point value
  - Description
  - Recommendation text
- Eco-swap data stored in structured format:
  - Traditional option (name, cost, image)
  - Sustainable alternative (name, cost, image)
  - Style tip
  - Environmental impact note
- Venue data pulled from venue database if available
- Sustainable planning guide PDF pre-created

### Accessibility
- Scorecard toggle switches keyboard accessible (space to toggle)
- ARIA labels on all interactive elements
- Card flip accessible via keyboard (Enter to flip)
- Chart data available as text alternative for screen readers
- Color contrast WCAG AA compliant
- Focus indicators visible
